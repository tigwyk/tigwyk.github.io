Public Banano Nodes

2021-04-26 :: Tigwyk
#banano  #crypto 

Something I noticed recently while hanging out in on some Banano-related discord servers was that there didn’t appear to be a reliable list of the public Banano nodes. There were several requests from developers looking to connect to a node, so I had an idea. I forked SomeNano’s Nano Public Node site, and with a few minor changes… I created a Banano version!

Introducing… public.banano.live!

Link: https://public.banano.live
